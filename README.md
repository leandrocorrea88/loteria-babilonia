# loteria-babilonia
Projeto de aprendizado de Python, Git e GiHub introdutórios

## Objetivo
Criar uma loteria onde um número da sorte é escolhido e o usuário tenta acerta-lo

## Ferramentas a utilizar
1. Python, com a IDE VSCode
2. GitHub e Git, na IDE VSCode

## Dinâmica do projeto
O repositório foi criado direto no GitHub, porém toda a sua gestão será feita no VSCode usando as ferramentas integradas
